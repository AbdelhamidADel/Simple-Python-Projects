# About Project
This project is divided into two parts,
the first section is the presentation of the library with the books available in it with the cover of those books to attract those interested to the library
The second part is the library system by selecting the books to be borrowed with identification. It also includes a book return policy.
It is important to take into account the readers’ lover’s requests through the option to add books to the library’s book list
